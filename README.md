# نظام دعوات واتساب - الكلية التقنية بالأحساء

نظام متكامل لإرسال دعوات الفعاليات عبر واتساب، تتبع الردود، وعرض الإحصائيات.

---

## نظرة عامة

يتيح لك هذا المشروع:
- عرض دعوة احترافية عبر صفحة ويب
- جمع تأكيدات الحضور عبر نموذج إلكتروني
- متابعة الإحصائيات عبر لوحة تحكم تفاعلية
- إرسال الدعوات تلقائياً عبر واتساب (Twilio)

---

## الملفات

| الملف | الوصف |
|-------|-------|
| `whatsapp_invitation.html` | صفحة الدعوة الاحترافية |
| `attendance_form.html` | نموذج تأكيد الحضور |
| `dashboard.html` | لوحة متابعة الإحصائيات |
| `whatsapp_interactive_guide.html` | دليل أنواع رسائل واتساب |
| `whatsapp_sender.py` | سكريبت Python لإرسال الدعوات |
| `whatsapp_message.txt` | قالب رسالة الدعوة |
| `whatsapp_complete_guide.txt` | دليل شامل لـ Twilio و API |
| `INSTRUCTIONS.txt` | تعليمات الاستخدام |
| `QUICK_START_CURSOR.md` | دليل سريع لـ Cursor |
| `requirements.txt` | متطلبات Python |

---

## البدء السريع

### 1. اختبار صفحات HTML

```
1. افتح المشروع في Cursor أو VS Code
2. انقر يمين على whatsapp_invitation.html
3. اختر "Open with Live Server"
4. شاهد الدعوة في المتصفح
```

### 2. إرسال دعوات عبر واتساب

**يدوياً:**
1. افتح `whatsapp_message.txt`
2. انسخ الرسالة وأرسلها عبر واتساب

**تلقائياً:**
```bash
pip install twilio
python whatsapp_sender.py
```

### 3. متابعة الردود
- افتح `dashboard.html` في المتصفح
- استورد بيانات CSV أو تابع الردود المباشرة
