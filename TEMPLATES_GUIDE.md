# دليل القوالب المتعددة

## القوالب المتاحة

| القالب | الاستخدام | الحقول |
|--------|-----------|--------|
| **دعوة عامة** | للمدعوين العاديين | الاسم، رقم الجوال |
| **دعوة رسمية** | للمسؤولين وكبار الشخصيات | الاسم، رقم الجوال، المنصب |

---

## تفعيل القالب الرسمي (للمسؤولين)

### 1. إنشاء القالب في Twilio

```bash
python create_vip_template.py
```

سيعطيك **Content SID** (مثل: HXxxxxxxxx)

### 2. إرسال القالب لموافقة WhatsApp

- اذهب إلى [Twilio Console](https://console.twilio.com) → Messaging → Content
- ابحث عن القالب المنشأ
- اضغط **Submit for Approval** 
- انتظر الموافقة (عادة 24 ساعة)

### 3. إضافة القالب للنظام

**محلياً** - أضف في `config.json`:
```json
{
  "content_sid": "HX7f91572f7f87564aa0265dbe20b6ae12",
  "content_sid_vip": "HXxxxxxxxx"
}
```

**على Render** - أضف متغير البيئة:
```
CONTENT_SID_VIP=HXxxxxxxxx
```

## الاستخدام

### من لوحة التحكم

1. اختر **نوع الدعوة** من القائمة (دعوة عامة / دعوة رسمية)
2. للدعوة الرسمية: أدخل **المنصب** (مثل: مدير الإدارة، رئيس القسم)
3. أرسل

### الإرسال الجماعي - القالب الرسمي

صيغة كل سطر:
```
الاسم، الرقم، المنصب
```

مثال:
```
سعادة أحمد الحربي، 0551234567، مدير الشؤون الإدارية
سعادة محمد العتيبي، 0567891234، رئيس قسم الموارد البشرية
```
